Resources

My collection of favorite tools, resources, and links.

Learn

An investment in knowledge pays the best interest.

  • Udemy - An online learning and teaching marketplace with over 130,000 courses and 35 million students.
  • web.dev - A resource for developers to learn, share, and build web apps.
  • Duolingo - A platform that includes a language-learning website and mobile app, as well as a digital language proficiency assessment exam.
  • Unity Learn - A platform for learning how to use Unity.
  • patterns.dev - A collection of design patterns.
  • Wanago Blog - A blog about JavaScript, Angular, and Web Development.
  • Khan Academy - A non-profit educational organization created in 2008 by Salman Khan with the goal of creating a set of online tools that help educate students.
  • w3schools.com - A web development tutorial website.
  • Microsoft Learn - A free, online training platform that provides interactive learning for Microsoft products and more.
  • refactoring.guru - A collection of design patterns.
  • freecodecamp.org - A nonprofit community that helps you learn to code by building projects.
  • Developer Roadmaps - A set of charts demonstrating the paths that you can take and the technologies that you would want to adopt in order to become a frontend, backend or a devops.
  • System Design by Karan - Learn how to design systems at scale and prepare for system design interviews

    • Art

  • Pixiv - A Japanese online community for artists.
  • OpenArt - A AI image generator and discovery tool.
  • Behance - A social media platform owned by Adobe which claims "to showcase and discover creative work".
  • Unsplash - A website dedicated to sharing stock photography under the Unsplash license.
  • Sketchfab - A platform to publish, share, discover, buy and sell 3D, VR and AR content.
  • ArtStation - A showcase platform for games, film, media & entertainment artists.

    • Productivity

  • Ora - A project management tool for teams.
  • Notion - An all-in-one workspace for your notes, tasks, wikis, and databases.
  • Airtable - A spreadsheet-database hybrid, with the features of a database but applied to a spreadsheet.
  • ChatGPT - A chatbot that can talk about anything.
  • Money Lover - A personal finance management application.
  • Microsoft To Do - A cloud-based task management application.
  • Google Calendar - A time-management and scheduling calendar service.

    • Design

  • Figma - A collaborative interface design tool.
  • Canva - A graphic design platform.
  • Draw.io - A free online diagram software for making flowcharts, process diagrams, org charts, UML, ER and network diagrams.
  • Dribbble - A community of designers sharing screenshots of their work, process, and projects.
  • Excalidraw - A virtual collaborative whiteboard tool.
  • toools.design - A collection of tools for designers.
  • AI Design Resouces - A curated collection of AI tools, AI Trends & AI courses.

    • Software Development

    • Docker - A set of platform as a service products that use OS-level virtualization to deliver software in packages called containers.
    • DBeaver - A free multi-platform database tool for developers, database administrators, analysts, and all people who need to work with databases.
    • VS Code - A lightweight code editor.
    • Postman - A collaboration platform for API development.
    • GitKraken - A Git GUI client.
    • Resource.fyi - A collection of resources for developers.
    • The Public APIs List - A collective list of free APIs.
    • Free For Developers - A list of software (SaaS, PaaS, IaaS, etc.) and other offerings that have free tiers for developers.
